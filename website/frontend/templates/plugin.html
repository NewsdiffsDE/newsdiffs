Das Plugin für newsdiffs.de

Das newsdiffs.de Chrome Plugin zeigt Ihnen die Änderungen von Nachrichtenartikeln direkt im Browser an. Wenn sie das Plugin installiert und aktiviert haben, wird auf jeder Seite, die in unserem Archiv gespeichert ist, das newsdiffs.de Logo angezeigt. Wenn Sie auf dieses Logo klicken, bekommen Sie in einem Popup eine Auswahl zu den verschiedenen Versionen des Artikels angezeigt. Sobald Sie eine Version auswählen, wird Ihnen diese direkt auf der Seite angezeigt. Wollen Sie zum originalen Artikel zurückkehren, müssen Sie nur auf "Original Artikel anzeigen" klicken. Das newsdiffs.de Logo wird nicht angezeigt, wenn Sie auf einer Seite surfen, die NICHT Teil unseres Archivs ist.

Um das Chrome Plugin zu installieren, führen Sie bitte die folgenden Schritte aus:

Schritt 1

Laden Sie sich das Plugin hier herunter

Schritt 2

Rufen Sie in Ihrem Chrome Browser die folgende URL auf: chrome://extensions

Schritt 3

Klicken Sie auf "Erweiterung laden"

Schritt 4

Rufen Sie eine URL aus unserem Archiv auf

(Grundlage für unser Plugin war https://github.com/zthomae/newsdiffs-chrome.)
