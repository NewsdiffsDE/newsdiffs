Das Plugin für newsdiffs.de

Das newsdiffs Chrome Plugin zeigt Ihnen die Änderungen von Nachrichtenartikeln direkt im Browser an. Wenn sie das Plugin installiert und aktiviert haben, wird auf jeder Seite, die in unserem Archiv ist, das newsdiffs.de Logo angezeigt. Wenn Sie auf dieses Logo klicken, bekommen Sie im Popup eine Auswahl zu den verschiedenen Versionen des Artikels angezeigt. Sobald Sie eine Version auswählen, wird Ihnen diese direkt auf der Seite angezeigt. Wollen Sie zum originalen Artikel zurückkehren, müssen Sie nur auf "Original Artikel anzeigen" klicken. Das newsdiffs.de Logo wird nicht angezeigt, wenn Sie auf einer Seite surfen, die NICHT Teil unseres Archivs ist.

Um das Chrome Plugin zu installieren, führen Sie bitte die folgenden Schritte aus:

1. Laden Sie sich das Plugin herunter

2. Rufen Sie in ihrem Chrome Browser die folgende URL auf: chrome://extensions

3. Klicken Sie auf "Erweiterung laden"

4. Rufen Sie eine URL aus unserem Archiv auf
