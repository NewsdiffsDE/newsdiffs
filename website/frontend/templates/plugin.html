Plugin

So Kinders hier nen Proof of Concept für das Browser Plugin. Basiert auf https://github.com/zthomae/newsdiffs-chrome. Das Plugin funktioniert nur mit dem Chrome Browser und auch nur mit folgenden Seiten: bild.de, focus.de, spiegel.de und stern.de

Zur Installation die folgenden Schritte ausführen:

1. Zip entpacken
2. chrome://extensions/ aufrufen
3. Auf „Entpackte Erweiterung laden…“ klicken
4. Den entpackten Ordner auswählen
